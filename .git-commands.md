# Git Commands for Push/Pull

## Quick Commands

### Push all changes to GitHub
```powershell
cd C:\Project\pokemon_app
.\git-push.ps1 -Message "Applied teacher feedback: improved error logging, extracted inline code, documented CSP"
```

Or manually:
```powershell
cd C:\Project\pokemon_app
git add .
git commit -m "Applied teacher feedback: improved error logging, extracted inline code, documented CSP"
git push origin main
```

### Pull latest changes from GitHub
```powershell
cd C:\Project\pokemon_app
.\git-pull.ps1
```

Or manually:
```powershell
cd C:\Project\pokemon_app
git pull origin main
```

## First Time Setup (if not already done)

```powershell
cd C:\Project\pokemon_app
git init
git branch -M main
git remote add origin https://github.com/zo3f/pokemon_app.git
git add .
git commit -m "Initial commit"
git push -u origin main
```

## Check Status

```powershell
git status
```

## View Recent Commits

```powershell
git log --oneline -10
```
